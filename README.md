# Beer Style Statistical Analysis

This project uses Python to analyze beer data and answer two statistical questions involving alcohol by volume and state-level IPA proportions.

## Overview
The project compares the mean alcohol by volume (ABV) of American IPAs and American Pale Ales using an independent samples t-test. It also compares the proportion of American IPAs in California and Colorado using a two-proportion z-test.

## Methods Used
- Data cleaning with pandas
- Summary statistics
- Boxplot visualization
- Welch’s two-sample t-test
- Two-proportion z-test
- 95% confidence intervals

## Key Findings
- American IPAs had a significantly higher mean ABV than American Pale Ales
- California had a significantly higher proportion of American IPAs than Colorado
- Confidence intervals supported both hypothesis test conclusions
